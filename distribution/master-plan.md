# DISTRIBUTION MASTER PLAN (Influentia)

## Overview
Four pillars that compound over time. Each output is a permanent asset that works while we sleep.

## PILLAR 1: Programmatic SEO (~25 pages, evergreen)
**Goal:** Own every long-tail keyword around buyer intent, outreach, and lead intelligence.

**Strategy:** Create specific landing pages at `/landing/seo/[keyword].html` targeting high-intent, low-competition searches. Each page:
- ~800-1200 words of genuinely useful content (not keyword stuffing)
- Real examples and frameworks
- Internal links to main site
- Structured JSON-LD data
- Fast, clean HTML (no JS dependencies)

**Target keyword clusters:**
1. `[industry] + buyer intent detection`: "SaaS buyer intent", "agency lead intelligence", "B2B buyer signals"
2. `[competitor] + alternatives`: "alternatives to Expandi", "Lemlist alternative", "Apollo.io outreach replacement"
3. `how-to + problem`: "how to find people complaining about your product", "LinkedIn outreach without getting banned", "scale cold outreach without spam"
4. `specific solution`: "Reddit intent monitoring", "LinkedIn signal detection", "AI outreach personalization"
5. `niche-specific`: "B2B SaaS lead discovery", "agency client acquisition AI", "consultant lead intelligence"

## PILLAR 2: Blog Content (1 post/day, evergreen)
**Goal:** Build topical authority so Google/LLMs see us as THE expert on buyer intent/outreach.

**Strategy:** Each post solves one real problem, uses specific examples, links to Influentia naturally:
- ~1500-2000 words
- Real frameworks and examples (not fluff)
- Target one primary keyword + 2-3 secondary keywords
- Include author schema (Ermo, founder)
- Cross-link between posts for topical authority

**Content themes rotation:**
1. **Anti-volume philosophy** — Why most outreach fails
2. **Buyer intent frameworks** — How to spot signals
3. **Tool comparison/analysis** — Honest breakdowns
4. **Case studies** — Real results (use anonymized examples from early users)
5. **Industry analysis** — What's happening in outreach right now

## PILLAR 3: Social Content Drafting (3 formats/day)
**Goal:** Have ready-to-post content for every platform, find active conversations to join.

**Daily output:**
1. **Reddit post or comment** — Find 3 relevant threads daily. Write genuine help, mention Influentia only if valuable. Save as draft. Auto-post to r/SaaS build-in-public twice per week.
2. **X/Twitter thread** — 3-5 tweets, one sharp take. Hook on tweet 1. Value-first, soft CTA. Save as draft file.
3. **LinkedIn post** — Founder perspective, insight-driven, 5-8 paragraphs. Save as draft.

**Tone rules:**
- NEVER promotional, ALWAYS helpful
- Use specific numbers, not vague claims
- Sound like a founder who's been through it
- "We learned this the hard way" > "Our platform is amazing"
- No buzzwords (AI-powered, revolutionary, game-changing)

## PILLAR 4: AI Directory Submissions (first batch, then monitor)
**Goal:** Get Influentia listed in every directory that feeds ChatGPT, Gemini, Perplexity recommendations.

**Priority directory list (free, high-impact):**
1. There's An AI For That (theresanaiforthat.com)
2. AI Tool Directories (ai.io, aicrawler.com)
3. Futurepedia (futurepedia.io)
4. Toolify.ai
5. TopAI.tools
6. Opentools.ai
7. 100AI.tools
8. AI Scout
9. Product Hunt (self-submit)
10. AlternativeTo (submit as alternative to Expandi, Lemlist)

**Strategy:** Submit with unique descriptions, not copies of the homepage. Each submission should sound natural and highlight the unique angle (signal-based, not volume-based).

## EXECUTION PRIORITY
1. **Week 1:** Create 5-7 programmatic SEO pages + submit to 3 AI directories + set up blog + 3 days of social drafts
2. **Week 2:** Create 10-15 more SEO pages + submit to 5 more directories + keep social going
3. **Week 3:** Create remaining SEO pages + monitor rankings + double down on what's working
4. **Ongoing:** 1 blog post/day, social drafts, find and respond to 1-2 Reddit threads daily

## AUTOMATION RULES
- Every generated piece is saved with date and platform tags
- Never duplicate content across platforms verbatim
- All content must be genuinely helpful — no spam
- Measure what gets engagement, double down
- If something sounds like marketing, rewrite it
